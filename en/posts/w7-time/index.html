Windows 7 and time

27/Aug 2016
know-how windows

Some time ago I encountered an unordinary problem: I had to set a specific time in Windows 7 in order to run certain app. Unfortunately, Windows weren’t happy enough to allow me to do so, constantly changing time.

How I fixed it?

@echo off
:loop
date 12-07-27
time 14:24:00
goto loop

Don’t do it.

And what was the reason? Windows 7 in this particular situation didn’t allow me to change time - it was still changing, no matter what I had done - so, such a dirty trick was a way to do what I want to do.
